Инструкция по использованию оконного приложения, отображающего работу алгоритма машинного обучения "RandomForestClassifier" для предсказания оттока клиентов банка (Client-Churn prediction). 
1.Запустить приложение отправленное учеником в телеграмм
2. Дождаться запуска приложения
3. После того как приложение запустилось, ввести личные и финансовые данные клиента необходимые для предсказания 

4. После введения всех параметров, нажать на кнопку "РАССЧИТАТЬ ПРОГНОЗ"
